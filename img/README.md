# Рекламные баннеры для слота "o block slot"

## Инструкция по настройке рекламных баннеров

### 1. Создание папки
Папка `img` уже создана в корневой директории проекта.

### 2. Текущий статус
✅ **Файл `ad1.png` уже существует** - рекламный баннер готов к использованию!

### 3. Проверка отображения
После загрузки страницы баннер должен отображаться:
- В верхней части сайта
- С размером 728×90px
- С темным фоном для контраста
- Со скругленными углами

### 4. Поддерживаемые форматы
Слот поддерживает следующие форматы изображений:
- `ad1.png` (основной формат) ✅ **АКТИВЕН**
- `ad1.jpg` (альтернативный формат)
- `ad1.gif` (для анимированной рекламы)
- `ad1.webp` (современный формат)

### 5. Характеристики баннера
- **Размер**: 728×90px (стандартный leaderboard)
- **Максимальный размер**: 728px ширина, 90px высота
- **Формат**: PNG с прозрачностью или JPG
- **Вес**: до 150KB для быстрой загрузки

### 6. Расположение
- **Позиция**: правая боковая панель ✅ **ОПТИМАЛЬНО**
- **Тип**: Встроенный в интерфейс игры
- **Выравнивание**: по центру панели
- **Фон:** прозрачный, интегрированный в дизайн
- **Отступы**: гармоничные с другими элементами
- **Интеграция**: часть UI игры

### 11. Замена баннера
Для замены рекламного баннера:
1. Подготовьте новое изображение (728×90px)
2. Переименуйте в `ad1.png`
3. Замените файл в папке `img`
4. Обновите страницу для просмотра

### 12. Дополнительные возможности
Можно добавить несколько баннеров:
- `ad1.png` - верхний баннер (текущий) ✅
- `ad2.png` - можно добавить для нижнего баннера
- `ad3.png` - для боковых панелей

### 13. Техническая реализация
```html
<div class="ad-banner-top">
  <img src="img/ad1.png" alt="Advertisement" 
       onerror="this.style.display='none';this.parentElement.style.display='none'">
</div>
```
